## css selectors

1. global selector (\*)
2. id selectors (#)
3. class selectors (.)
4. tag selector

### id selector

```

#elementId {
    property: value;
}


```

### class selector

```
className {
    property: value;
}

```

### tag selector

```
tagName {
    property: value;
}
```
